## Docker Compose
The defined [docker-compose.yaml](docker-compose.yaml) has the following services:

- zookeeper
- kafka-broker1
- kafka-broker2
- schema-registry
- ksqldb-server
- ksqldb-cli
- kafka-connect
- kafka-rest
- ksql-datagen
- control-center
- minio
- postgres
- pgadmin

## CLI
In order to run the CLI just execute the following command:
```shell
python cli.py
```
